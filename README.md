# 🛠️ Digitools

<!-- ![Digitools Banner](https://i.ibb.co.com/R4kYfbCD/banner-Section.png)  -->

**Digitools** is a modern, fully responsive React web application that simulates a sleek digital tools & SaaS marketplace. Built with a beautiful Figma-inspired design, it features smooth product browsing, a fully functional shopping cart, live cart count, and elegant toast notifications.

### 🔗 [Live Demo](https://digitools-platform-lingkon.netlify.app/)

Perfect showcase of professional frontend development with interactive toggling between Products and Cart views.

---

## ✨ Key Features

1.  **🛍️ Interactive Product Showcase**
    Browse premium digital tools in a clean 3-column responsive grid. Each card shows name, description, pricing (with period), tags (Popular, New, Best Seller), feature list, icon, and Buy Now button.

2.  **🛒 Smart Cart System**
    Add products instantly → view in dedicated cart panel → remove individual items → live cart count in navbar → "Proceed to Checkout" clears everything with success feedback.

3.  **🎉 Premium UX with Toast Notifications**
    Powered by **React-Toastify** for beautiful alerts on Add to Cart, Remove, and Checkout.
    Central toggle buttons (Products / Cart) with default Product view.
    Fully responsive across mobile, tablet, and desktop.

**Bonus Features Implemented:**
* Cart count badge updates live in Navbar
* "Added to Cart" temporary feedback on Buy Now buttons
* Beautiful empty cart state
* JSON-powered product data (8 products with id, name, price, period, tags, features, icon)

---

## 🛠️ Technologies Used

* **React.js** (Functional Components + Hooks)
* **Tailwind CSS** + **DaisyUI**
* **JavaScript (ES6+)**
* **React-Toastify** (NPM package)
* **JSON** (for product data)
* Fully responsive design
